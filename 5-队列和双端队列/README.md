### 队列和双端队列

- 队列
```
队列是遵循先进先出(FIFO,先来先服务)原则的一组有序的列。
队列在尾部添加新元素，并从顶部移除元素。
最新添加的元素必须排在队列的末尾。

例子：常见的就是打印程序的队列了，需要打印5份文件，从1/5开始，一直到5/5结束，
如果中途添加了打印文件的话，会直接加到末尾，直到打印完所有的文件
```

- 双端队列数据结构
```
双端队列（dequ/double-ended queue）是一种允许我们同时从前端和后端添加和移除元素的特殊队列

例子：电影院排队，一个刚买了票的人忘记了还要咨询一些信息，直接回到头部问完走人，
队尾的人发现排队的人太多了，不想看了，也可以直接走人

在计算机科学中，双端队列的常见场景是存储一系列的撤销操作
每当用户在软件中进行了一个操作，该操作会被存在一个双端队列中
当用户点击撤销按钮时，该操作会被从双端队列中弹出，表示它被从后面移除了，
最先进行的操作完成后会被从双端队列的前端移除，

由于双端队列同时遵守了先进先出和后进先出原则
所以可以说它是把队列和栈相结合的一种数据结构

```
